Courriels en attente

Vérification des boîtes mail POP3 et IMAP

Click for English

OBJET

Vérification périodique des courriels sur les boîtes POP3 et IMAP. Ce programme remplace MailAttente qui est obsolète.


Cliquer sur les boutons de cette image pour obtenir la description de son action.

INSTALLATION

Extraire l'archive et exécuter InstallMailsInBox.exe, puis suivre les instructions du programme d'installation.

UTILISATION

Dans des conditions normales, Courriels en attente peut être minimisé dans la zone de notification. Il ne peut être fermé que par un clic doit sur son icone et le menu Quitte, ou dans sa fenêtre principale avec un double clic sur le bouton Quitte.
Par défaut, l'interface standard de "Courriels en attente" est une barre de boutons. La taille des boutons peut être réduite dans les préférences du programme.
Pour afficher un menu, placer le curseur de la souris dans la barre de boutons et cliquer sur le bouton droit, puis sélectionner "Afficher le menu".

Barre de boutons


Importation de comptes

Permet l'importation de comptes de notre précédent programme "MailAttente", ou importer des comptes "Outlook" (2007-2013) et "Thunderbird".
Le programme retrouve automatiquement les comptes disponibles dans ces applications. Sélectionner le type de compte dans la liste à gauche, le ou les comptes à importer.
Les données utiles du compte sélectionné s'affichent dans la liste à droite.
Cliquer ensuite sur le bouton "OK" pour valider l'importation ou "Annuler" pour y renoncer.
Dans le cas d'"Outlook" et de "Thunderbird", les mots de passe ne sont pas importés et vous devez les mettre à jour en éditant le(s) compte(s) correspondant(s).



Affiche le journal d'activité du compte

Selon les options générales du programme, il est possible de sauvegarder le journal, le journal d'activité courant s'ajoutant à celui enregistré précédemment.



Vérifie les boîtes de courriel de tous les comptes actifs

Pour rendre un compte de courriel actif ou inactif, il faut cocher la case correspondante lors de la création ou la modification d'un compte de courriel.



Vérifie la boîte de courriel du compte sélectionné s'il est actif.



Lance le client courrier sélectionné par l'utilisateur

Si ce client courrier est connu du programme, l'icone de ce client courrier peut figurer sur ce bouton. Par exemple, Google Mail, Thunderbird, Windows Mail, Outlook.



Supprime le compte sélectionné

Une confirmation est demandée, cette opération est irréversible (enfin presque, une sauvegarde de la dernière liste de comptes existe dans le dossier utilisateur de Courriels en attente).



Ajoute un nouveau compte à vérifier

Compléter les différents champs de cette boîte de dialogue. Les paramètres adéquats sont donnés par le fournisseur du service de courrier.
Les protocoles acceptés par le programme sont POP3 et IMAP.
Les mots de passe sont stockés chiffrés dans le fichier de configuration du programme.


Modifie le compte sélectionné

La boîte de dialogue est similaire à celle de l'ajout d'un nouveau compte. Les champs sont préremplis avec les paramètres enregistrés du compte.


Ouvre la fenêtre de modification des options générales de Courriels en attente.

Lancer au démarrage du système: cette option permet de vérifier automatiquement les boîtes de courriel dès le démarrage du système. Dans tous les cas, lors de l'arrêt dus système, la configuration en cours est sauvegardée.
Démarrage minimisé: La fenêtre principale du programme ne s'affiche pas au démarrage. Pour l'afficher, cliquer simplement sur son icone dans la barre des tâches ou dans la zone de notification.
Sauvegarder taille et position de la fenêtre: La taille de la fenêtre, les dimensions des volets, la largeur des colonnes sont sauvegardées à la fermeture du programme.
Minimiser au lancement du client Courrier: La fenêtre est réduite en icone lorsque l'on clique sur le bouton de lancement du client Courrier.
Restaurer la fenêtre si nouveaux messages: Lors de l'arrivée de nouveaux messages, la fenêtre du programme apparaît à l'écran.
Masquer l'icône dans la barre des tâches: Lorsque le programme est minimisé, l'icône dans la barre des tâches n'est pas affichée. Cliquer avec le bouton gauche de la souris pour afficher la fenêtre du programme et avec le bouton droit de la souris pour afficher un menu des commandes essentielles.
Ne pas rechercher de mise à jour: Utiliser lorsque la machine de dispose pas de connexion Internet ou lorsque l'abonnement limite la quantité de données. Lorsque cette case est cochée, le programme recherche automatiquement une éventuelle mise à jour tous les 7 jours.
Conserver le journal du compte: Par défaut, le journal d'activité n'est conservé que jusqu'à la session suivante. En sélectionnant cette option, il est possible de sauvegarder le journal, le journal d'activité courant s'ajoutant à celui enregistré précédemment.
Ne pas afficher l'alerte d'arrêt du programme: Par défaut, une boîte d'alerte est affichée lors d'un clic sur le bouton d'arrêt du programme. Cocher la case dans cette boîte d'alerte ou dans ce dialogue pour ne plus afficher cette alerte. Il faut alors un double clic sur le bouton "Quitter Courriels en attente" pour quitter le programme.
Ne pas afficher l'alerte de masquage de la fenêtre: Par défaut, une boîte d'alerte est affichée lorsque l'on tente de fermer le programme en utilisant les contrôles de fermeture du système ((boutons ou menu dans la barre de titre, icône dans la barre des tâches). Cocher la case dans cette boîte d'alerte ou dans ce dialogue pour ne plus afficher cette alerte.
Activer les notifications: Par défaut, les notifications du programme au système ne sont pas activées. Cocher cette case pour que l'arrivée de nouveaux courriels apparaissent dans la zone de notifications.
Afficher les messages de tous les comptes: Par défaut, les courriels du compte sélectionné sont affichés dans la liste des courriels. Activer cette option pour afficher l'ensemble des messages en attente.


Affiche le journal d'activité de Courriels en attente.


Affiche une aide du programme (ce document).


Affiche la boite de dialogue A propos

Cliquer sur "Recherche de mise à jour" pour savoir si une nouvelle version de Courriels en attente" est disponible.
Cliquer sur "Web site" pour visiter le site de l'auteur du programme.


Quitte Courriels en attente

Par défaut, une boîte d'alerte est affichée lors d'un clic sur ce bouton. Cocher la case dans cette boîte d'alerte ou dans ce dialogue pour ne plus afficher cette alerte. Il faut alors un double clic sur ce bouton pour quitter le programme.



Masque le programme avec les contrôles de fermeture du système

C'est à dire l'icône en haut à droite de la barre de titre de la fenêtre de Courriels en attente, le menu système en haut et à gauche de la barre de titre du programme ou le menu déroulant de l'icône du programme dans la barre des tâches.
Lors de l'utilisation d'un de ces contrôles, le programme en se ferme pas mais est minimisé.
Par défaut, une boîte d'alerte est affichée. Cocher la case dans cette boîte d'alerte ou dans ce dialogue pour ne plus afficher cette alerte.

Mails In Box

Check POP3 et IMAP mail boxes
Cliquer pour le Français

OBJECT
Periodic mails check on POP3 and IMAP mailboxes. The program replaces Mailattente which is obsolete.


Click a button on the image to obtain a description of its action.


INSTALLATION
Extract archive and run InstallMailInBox.exe, then follow instructions of the setup program.

USAGE
The program can be run at windows start and minimized, click the Settings button. To display a menu bar, place the mouse cursor in the buttons bar and right click the mouse, then select "Display the menu".

Buttons bar


Accounts import
Can import our previous program "MailAttente" accounts, or importer "Outlook" (2007-2013) and "Thunderbird" accounts. The program finds automatically the available accounts from these applications. Select the account type in top combo box, then the accounts to import in the left side list. In the right side list, main data of the selected account are displayed. Now, click the "OK" button to valid the import or the "Cancel" button to ignore the import. With "Outlook" and "Thunderbird", passwords are not imported and you will have to enter them by editing related account(s).



Displays selected account log
In general options, it is possible to save the log file, the current log is added at the end of the previously saved log.



Check all active accounts mails boxes
To activate a mail account, check the box "Active account" when you add or edit a mail account.



Check selected account mail if it is active.



Launch the selected mail client
If the program knows the mail client, its icon can be drawn on the button. For instance Google Mail, Mozilla Thunderbird, Windows Mail, Outlook.



Delete selected account
The user is asked to confirm, as this operation is irreversible (however, a backup of the last known accounts base is available in the Mails In Box application data folder).



Add a new account to check
Complete different fields of this dialog box. Proper parameters are given by the mail service provider. Accepted protocols are POP3 and IMAP.
Passwords are encrypted in "Mails In Box" settings file.


Edit selected account
The dialog box is similar to the "Add Account" Dialog box. Fields are filled with current account parameters.



Open the main settings dialog box
Launch at system startup: This option allow a mail boxes check on system startup. In all cases, at system logout, program settings are saved.
Start minimized: The program main window is not displayed at startup. To display this window, simply click the program icon in the task bar or the notification area.
Save window size and position: The window size, size of panels, columns widths are saved when user quit the program or logout the system.
Minimize on Mail client launch: The program window is minimized after a click on the "Launch mail client" button.
Restore program window on new messages: When new messages are found, the program window is displayed if it was hidden.
Hide program icon in task bar: When the program window is minimized, icon in taskbar is hidden. Click the left mouse button on the icon in the notification area and the right mouse button on the icon in notification area to display a menu with essential commands.
Do not check for update: Useful when there is no Internet connection or when Internet subscription limits data amount. If this box is checked, the program checks for update every 7 days.
Save account log: Only the current activity log for accounts and program is saved. Select this option to have a complete activity log, the current log is concatenated to the previous one.
Don't display exit program alert: By default, an alert box is displayed when the "Quit Mails In Box" button is clicked. Check the box in the alert box or in the settings dialog to no longer display the alert. On this case, a double click on the "Quit" button is needed to exit the program.
Don't display hide window alert: By default, an alert box is displayed when the user try to close the program using the system close controls (buttons or menu on the title bar, icon on the system taskbar). Check the box in the alert box or in this dialog to no longer display this alert.
Enable notifications: By default, program notifications to the system are disabled. Check this box and the new mails events will be displayed in the notification area.
Display all accounts mails: By default, only the selected account mails are displayed in the mails list. Enable this option to display all the mails checked in the list.


Display the "Mails In Box" complete activity log
In general options, it is possible to save the log file, the current log is added at the end of the previously saved log.


Display program help (this document).


Display the About dialog box
Click on "Search update" to know if a new version of "Mails In Box" is available. Click on "Web site" to visit the author's web site.


Quit Mails In Box
By default, an alert box is displayed when the button is clicked. Check the box in the alert box or in the settings dialog to no longer display this alert. A double click on the button is needed to quit the program.



Hide program window with the system close controls
IE the icon on the top right of "Mails In Box" title bar on the top right of the window, the menu on the top left of the title bar, or the menu of the program icon on the taskbar.
When using one of these controls, the program doesn't close but is minimized. By default, an alert box is displayed.Check the box in the alert box or in the settings dialog to no longer display this alert.

Use control panel's program management system to uninstall the program.
