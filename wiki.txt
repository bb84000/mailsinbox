# Mails In Box
[Vers la page d'accueil en Français ](https://github.com/bb84000/mailsinbox/wiki/Accueil)

**Mails In Box** is a program to periodically check if there are mails on POP3 and IMAP mailboxes.

![Mails In Box](https://github.com/bb84000/mailsinbox/raw/master/images/mailsinboxe.jpg)

### INSTALLATION
Extract archive and run InstallMailInBox.exe, then follow instructions of the setup program.

### USAGE
On first run, you must create new account(s), or import account from previous MailAttente or import account(s) from Outlook 2007-2013 and Thunderbird. In the two latter cases, passwords are not imported and you need to update them by editing the account(s).

The program can be run at windows start and minimized, click the Settings button.

In regular conditions, **Mails In Box** can run minimized in the system tray.
It can onbly be closed by right click on its icon and choose _Quit_, or in its main windows,
with _Quit_ button.

### UNINSTALL
Use control panel's program management system to uninstall the program.

### NOTICE
**Mails In Box** software can be used fee of charge.
Use the program at your own risks. Take care, mail delete on server is irreversible.

# Courriels en attente
[Go to English home page ](https://github.com/bb84000/mailsinbox/wiki/Home)

**Courriels en attente** est un programme qui vérifie périodiquement s'il y a des courriels sur les boîtes POP3 et IMAP

![Courriels en attente](https://github.com/bb84000/mailsinbox/raw/master/images/mailsinbox.jpg)

### INSTALLATION
Extraire l'archive et exécuter InstallMailsInBox.exe, puis suivre les instructions du programme d'installation.

### UTILISATION
Lors de son premier lancement, vous devez créer un ou plusieurs nouveaux comptes, ou importer des comptes de courrier de notre précédent programme MailAttente, ou importer des comptes Outlook et Thunderbird. Dans ces deux dernier cas, les mots de passe ne sont pas importés et vous devez les mettre à jour en éditant le(s) compte(s) correspondant(s).

Dans des conditions normales, **Courriels en attente** peut être minimisé dans la zone de notification. 
Il ne peut être fermé que par un clic doit sur son icône et le menu _Quitte_, ou dans sa fenêtre principale avec le bouton _Quitte_.

### DESINSTALLATION
Utiliser le système de gestion de programmes du panneau de configuration pour désinstaller le programme.

### NOTE
Le logiciel **Courriels en attente** peut être utilisé librement.
Utilisez **Courriels en attente** à vos risques et périls. Attention, l'effacement d'un message sur le serveur est irréversible.
